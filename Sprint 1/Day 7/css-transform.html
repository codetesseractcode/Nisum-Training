CSS Transform Explanation:

This example demonstrates three different CSS transform effects applied on hover:

  • Rotate: Uses transform: rotate(45deg) to rotate the element by 45 degrees when hovered.
  • Scale: Uses transform: scale(1.2) to increase the element's size by 20% when hovered.
  • Translate: Uses transform: translateY(-20px) to move the element 20px upward when hovered.

Each transform is applied with a smooth transition effect using transition: transform 0.3s ease, background-color 0.3s ease, which creates a gradual animation over 0.3 seconds.
