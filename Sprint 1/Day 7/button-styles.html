Button Styles Challenge

Button Styling Explanation:

This button demonstrates several advanced CSS styling techniques:

  • Rounded corners: Using border-radius: 50px to create a pill-shaped button.
  • Gradient background: Using linear-gradient to create a smooth color transition from purple to blue.
  • Hover animation: When hovered, the button:
    • Scales up slightly with transform: scale(1.05)
    • Changes to a slightly different gradient
    • Gets an enhanced shadow effect
  • Smooth transitions: All hover effects are smoothly animated using transition properties with a 0.3-second duration.

Additional styling features include uppercase text, letter spacing, and a slight scale-down effect when clicked.
