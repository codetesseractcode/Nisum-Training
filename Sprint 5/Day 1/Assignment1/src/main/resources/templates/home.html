Spring Security Demo

Welcome to Spring Security Authentication Demo

This application demonstrates both Basic and Form-based authentication using Spring Security.

Features:
  • Basic Authentication for REST APIs
  • Form-based Authentication for Web Interface
  • Role-based Authorization (USER, MANAGER, ADMIN)
  • In-memory User Store
  • Custom Login/Logout handling
