Responsive Gallery Explanation:

This image gallery uses CSS Grid to create a responsive layout that adapts to different screen sizes:

  • On desktop (>768px): 3 columns
  • On tablet (480px-768px): 2 columns
  • On mobile (<480px): 1 column

Additional features include:

  • Hover effects with image scaling
  • Caption overlay that appears on hover
  • Consistent image sizes using object-fit
  • Rounded corners and subtle shadow effects
